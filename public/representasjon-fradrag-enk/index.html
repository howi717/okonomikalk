Representasjon fradrag i enkeltpersonforetak

Representasjon kan være aktuelt når kostnaden gjelder kunder, samarbeidspartnere eller forretningsmessige møter. Det er samtidig et område der det er viktig å være nøyaktig.

Se hvordan dette er for deg

Representasjon

Hva regnes som representasjon?

Representasjon gjelder normalt utgifter knyttet til forretningsmessige møter, kundepleie eller lignende, men reglene har begrensninger.

  • Noter hvem som deltok og formålet.
  • Ta vare på kvittering.
  • Vurder beløpsgrenser og type servering.

Forskjell

Representasjon eller vanlig reiseutgift?

Mat på reise og representasjon kan ligne, men er ikke alltid det samme. Vurder hva kostnaden faktisk gjelder.

Spørsmål

Ofte stilte spørsmål

Kan jeg føre all representasjon?

Nei. Det finnes begrensninger, og kostnaden må være forretningsmessig begrunnet.

Må jeg dokumentere hvem som var med?

Ja, det er lurt å dokumentere deltakere og formål.

Hvorfor bør dette skilles fra mat på reise?

Fordi reglene og vurderingen kan være ulike.

Forbehold

Viktig å vite

ØkonomiKalk gir generell informasjon og enkle beregninger. Faktiske tall og regler kan variere. Ved usikkerhet bør du kontrollere opplysningene med offentlige kilder, regnskapsfører eller revisor.
