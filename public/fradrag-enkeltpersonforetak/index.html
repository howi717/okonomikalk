Fradrag enkeltpersonforetak

Lurer du på hva du kan føre som kostnad i enkeltpersonforetaket? Denne siden gir en enkel oversikt over typiske fradrag og leder deg videre til kalkulatoren der du kan teste egne tall.

Se hvordan dette er for deg

Oversikt

Vanlige kostnader å vurdere

Typiske kostnader kan være utstyr, verktøy, rekvisita, regnskapsprogram, tjenester, mobil/internett, reise, overnatting, bilrelaterte utgifter, programvare og markedsføring.

  • Kostnaden bør være knyttet til virksomheten.
  • Ta vare på kvitteringer og dokumentasjon.
  • Ved tvil bør du sjekke med regnskapsfører eller Skatteetaten.

Dokumentasjon

Hvorfor dokumentasjon er viktig

Fradrag handler ikke bare om å vite hva som kan trekkes fra. Du bør også kunne dokumentere hva kostnaden gjelder, når den ble betalt, og hvorfor den hører til virksomheten.

Kalkulator

Test fradragene med egne tall

I næringsverktøyet kan du legge inn flere kostnadstyper og se hvordan de påvirker resultat, skatt og netto etter skatt.

Spørsmål

Ofte stilte spørsmål

Kan jeg føre alt jeg kjøper som fradrag?

Nei. Kostnaden må være relevant for virksomheten. Private kostnader skal normalt ikke føres som næringskostnad.

Må jeg ha kvittering?

Du bør ha dokumentasjon på kostnaden. Kvitteringer, fakturaer og kontoutskrifter er nyttig.

Kan jeg bruke siden før jeg sender til regnskapsfører?

Ja. Siden er nyttig for å få oversikt før du sender tall videre eller spør om råd.

Forbehold

Viktig å vite

ØkonomiKalk gir estimater og generell informasjon. Faktiske skattetall kan avvike. Ved usikkerhet bør du kontrollere tallene med Skatteetaten, regnskapsfører eller revisor.
