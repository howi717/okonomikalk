Utstyr fradrag i enkeltpersonforetak

Kjøper du utstyr, verktøy, PC, rekvisita eller tjenester som brukes i virksomheten, kan kostnaden være relevant i oversikten. ØkonomiKalk hjelper deg å se hvordan slike kostnader påvirker resultatet.

Se hvordan dette er for deg

Utstyr

Hva slags utstyr kan være relevant?

Dette kan være PC, verktøy, rekvisita, arbeidsklær som ikke er private klær, regnskapsprogram, underleverandører eller andre tjenester.

  • Kostnaden bør brukes i virksomheten.
  • Ta vare på kvittering eller faktura.
  • Større innkjøp kan ha egne regler for fradrag/avskriving.

Blandet bruk

Privat og næring

Hvis noe brukes både privat og i virksomheten, bør du vurdere næringsandelen og ikke føre alt ukritisk.

Spørsmål

Ofte stilte spørsmål

Kan PC føres som kostnad?

Det kan være relevant hvis PC-en brukes i virksomheten, men vurder bruk, pris og dokumentasjon.

Kan arbeidsklær føres?

Arbeidsklær kan være relevant når de ikke er vanlige private klær.

Hva med underleverandører?

Tjenester kjøpt inn til virksomheten kan være relevante kostnader når de er dokumentert.

Forbehold

Viktig å vite

ØkonomiKalk gir generell informasjon og enkle beregninger. Faktiske tall og regler kan variere. Ved usikkerhet bør du kontrollere opplysningene med offentlige kilder, regnskapsfører eller revisor.
